# Bài 1: Quản lý Sinh viên

## URL Truy cập
- Trang chủ: `http://localhost:8080/Labs/project_lab09/bai1/public/?c=student&a=index`

## Cấu hình Database
- Host: `localhost`
- Database Name: `it3220_php`
- Username: `root`
- Password: *(trống nếu dùng XAMPP mặc định)*
- Charset: `utf8mb4`

## Cách Import SQL
1. Mở phpMyAdmin tại `http://localhost:8080/phpmyadmin`.
2. Tạo database mới với tên `it3220_php`.
3. Import file SQL: `database/it3220_php.sql`.
4. Kiểm tra các bảng `students`, `books`, `members` đã được tạo thành công.
